3D Modeling Project

Furniture Design Collection

Project Overview

Created a series of modern furniture pieces using SketchUp and rendered with V-Ray. This collection showcases my ability to design functional and aesthetically pleasing furniture with attention to ergonomics and material properties.

Project Details

  • Client: Personal Project
  • Year: 2023
  • Software: SketchUp, V-Ray, AutoCAD
  • My Role: 3D Modeler & Designer

Furniture Collection

Modern Lounge Chair

Ergonomic design with walnut finish and leather upholstery.

Dining Table

Extendable table with powder-coated steel base.

Sectional Sofa

Modular design with multiple configuration options.

Home Office Desk

Compact workspace with integrated cable management.

Modeling Process

1. Concept Sketching

Created initial sketches to explore form and function.

2. Technical Drawings

Developed precise measurements and joinery details in AutoCAD.

3. 3D Modeling

Built the digital models in SketchUp with proper scale and proportions.

4. Material Application

Applied realistic materials and textures to the models.

5. Lighting Setup

Configured studio lighting for optimal product presentation.

6. Final Rendering

Produced high-quality renders using V-Ray.

Technical Specifications

Polygon Count

15,000 - 20,000 polys per model

Render Time

Approx. 45 minutes per image (4K resolution)

Texture Resolution

2048x2048 for main materials

File Formats

SKP, DWG, OBJ, FBX
